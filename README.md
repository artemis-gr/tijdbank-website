# Tijdbank website

Static website for the Tijdbank project.

## Tech
- Plain HTML, CSS and vanilla JavaScript
- No build step, no backend
- Self-hosted fonts and images

## Usage
- Entry point: `index.html`
- Can be served by any static web server (Apache, Nginx...)

## Notes
- Section background colors are controlled via `data-bg` attributes
- Header color updates on scroll via `js/main.js`
